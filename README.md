# Beacon Alike - Custom Link-in-Bio

A beautiful, responsive, and customizable "Link-in-Bio" page for your social media profiles. Host it yourself and simplify your online presence.

## Features
-   **Responsive Design**: Works perfectly on mobile, tablet, and desktop.
-   **Customizable Links**: Easily add, remove, or update links.
-   **Social Media Icons**: Integrated icons for major platforms.
-   **Fast & Lightweight**: Built with pure HTML, CSS, and vanilla JavaScript (no heavy frameworks).

## Usage
1.  **Clone the repo**:
    ```bash
    git clone https://github.com/yourusername/beacon-alike.git
    ```
2.  **Customize Content**:
    -   Open `index.html`.
    -   Update the profile image source, name, and bio text.
    -   Modify the `<a>` tags in the links section to point to your own URLs.
3.  **Host**:
    -   Upload the files to any static hosting service (Hostinger, GitHub Pages, Netlify, Vercel).
